# AutoHopper Usage Guide

## Basic Usage

1. Hold a stick.
2. Right-click a hopper.
3. Sneak on the hopper.
4. Inventory items transfer automatically.

---

## Common Uses

### Survival Storage Systems

Quickly dump mined or farmed items into storage systems.

### Mob Farms

Transfer inventory items into hopper systems faster.

### Community Servers

Improve inventory management and gameplay flow.

---

## Permission Example

```text
/lp group default permission set autohopper.use true
```
