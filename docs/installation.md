# AutoHopper Installation Guide

## Requirements

- Minecraft Java Edition 1.17+
- Paper or Spigot server
- Java installed on the server

---

## Installation Steps

1. Download the latest AutoHopper release.
2. Place the `.jar` file into your server `plugins` folder.
3. Restart the server.
4. Verify the plugin loads correctly.

---

## Permissions

### Default Permission

```text
autohopper.use
```

### LuckPerms Example

```text
/lp group default permission set autohopper.use true
```

---

## Commands

```text
/autohopper enable
/autohopper disable
```
